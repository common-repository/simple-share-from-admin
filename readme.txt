== Description ==

Adds simple manual share links for Twitter and Facebook to the list of posts admin page.

Why another Twitter/Facebook sharing plugin? All the other were just too fancy or automatic. For an explanation see the post at:
http://www.moscowcoffeereview.com/programming/simple-share-from-admin-wordpress-plugin/

== Installation ==

1. Upload `simple-share-from-admin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on Posts on the main menu to see a list of all your posts. The new links should appear in in this table.
